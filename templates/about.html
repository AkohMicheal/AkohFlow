About Task Management App

This is a simple and clean task management app built to help you stay on top of your to-do list without the clutter. Whether you’re juggling school, work, or personal projects, this app lets you create tasks, set priorities, and mark things done—so you can feel more organized and less stressed.

It’s responsive, easy to use, and built with modern tech to work on any device. Perfect for anyone who wants to stop forgetting things and start getting stuff done 💪.

We are constantly working on improving the app and adding new features based on user feedback.
